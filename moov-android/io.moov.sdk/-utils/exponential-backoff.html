exponentialBackoff

@JvmStatic
fun exponentialBackoff(base: Duration, attemptNum: Int, multiplier: Int, max: Duration?): Duration

Calculates the next exponential backoff duration for the given parameters

The backoff is calculated using the following formula:

backoff = (base * multiplier ^ attemptNum) +/- jitter
Content copied to clipboard

jitter is a randomly generated amount ranging from 0, base/2. If max is provided, the backoff will be capped at max +/- jitter.

Parameters

base

the base backoff duration

attemptNum

the attempt which is being calculated

multiplier

an arbitrary multiplier to use for backoff calculation

max

optional value to clamp the maximum duration at
